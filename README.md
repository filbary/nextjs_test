# Next Example

นี่คือ code ที่ใช้ใน Video หัวข้อ "รู้จักกับ Next.js 14 แบบ Quick Overview"

สามารถ run ได้ทันที่ด้วยคำสั่งนี้เพื่อทำการ run project ขึ้นมา

```shell
npm install # ลง package
npm run dev # run project
```
